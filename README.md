# curso-nestjs

- Instalando o Nest na maquina:
npm install -g @nestjs/cli

- criando um projeto:
nest new livros-api

- Inicializa o servidor:
cd livros-api
npm run start

- Inicializa o servidor no modo dev que atualiza os dados em tempo real
npm run start:dev
